# Default gamemode

## Variable: `default-gamemode`

### `default-gamemode` is a variable used to store the gamemode after revival

Values it can be set to and their ratings **(CASE SENSITIVE)**:

1. SPECTATOR (⭐)
2. SURVIVAL (⭐⭐⭐)
3. CREATIVE (⭐)
4. **ADVENTURE (**⭐⭐⭐⭐⭐**)**
