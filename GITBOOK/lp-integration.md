**OUTDATED /** **UNTRANSLATED CONTENT**
{% endhint %}

If you're using LuckPerms and want to change permissions in the FDB, you need to check which permissions each command uses.

1. Look on [Github](https://github.com/sazenice/FakeDeathBan)
2. Open `src/main`
3. Open `resources`
4. Open `plugin.yml`

Here you'll see all the permissions (e.g., fakedeathban, undeathban)
