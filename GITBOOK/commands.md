# Commands

{% hint style="info" %}

<details>

<summary><strong>UnDeathBan</strong></summary>

Teleports all dead players to you and revives them.

**Permission:** OP

**Usage:** `/undeathban [player...]`

**Aliases:**

* udb
* undb
* udban
* udeathb
* undeathb
* undban
* udeathban

</details>

<details>

<summary><strong>DefaultGamemode</strong></summary>

**Permission:** OP

**Usage:** `/defg <gamemode>`

**Aliases:**

* defaultgamemode
* dgamemode
* defaultg
* dg

{% hint style="info" %}
