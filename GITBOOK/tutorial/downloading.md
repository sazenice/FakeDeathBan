# Downloading

Download plugin on [GITHUB](https://github.com/sazenice/FakeDeathBan/releases) or on [MODRINTH](https://modrinth.com/plugin/fakedeathban).

After that, simply put the plugin into the `plugins` folder, give the server a quick restart and it should be good to go
