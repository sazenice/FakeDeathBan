# Resetting

You can reset the plugin settings by deleting the `FakeDeathBan` folder in the `plugins` folder

{% hint style="warning" %}
Resetting will delete all data in the plugin, including players who have been deathbanned and frozen players
{% endhint %}

{% hint style="danger" %}
Never delete the contents of `config.yml` or `lang`. Always delete only the entire `FakeDeathBan` folder.
{% endhint %}
